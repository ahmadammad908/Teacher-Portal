API REFERENCE

Our RESTful API provides programmatic access to Hiderox services:

Authentication

  -H "Content-Type: application/json" \
  -d '{
    "api_key": "your_api_key",
    "timestamp": "current_timestamp",
    "signature": "generated_signature"
  }'

Create Privacy Session

curl -X POST "https://api.hiderox.com/session" \
  -H "Authorization: Bearer your_access_token" \
  -H "Content-Type: application/json" \
  -d '{
    "tier": "business",
    "duration": "30d",
    "services": ["vpn", "email", "storage"]
  }'
